# Fashion Recommendation and Purchase Behavior Analytics

## Project Overview

This project analyzes a fashion recommendation dataset to explore customer purchase behavior using exploratory data analysis (EDA), customer segmentation, and machine learning. The goal is to identify patterns related to customer demographics, product attributes, engagement metrics, and recommendation features.

## Objectives

* Analyze customer demographic characteristics.
* Explore product attributes associated with purchases.
* Evaluate user engagement behavior.
* Analyze product score features.
* Assess recommendation priority performance.
* Segment customers using K-Means clustering.
* Predict purchase behavior using machine learning models.

## Methods

* Exploratory Data Analysis (EDA)
* Correlation Analysis
* K-Means Clustering
* Logistic Regression
* Decision Tree
* Random Forest
* Gradient Boosting

## Tools

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Results

The analysis found only weak relationships between individual features and purchase behavior. Customer segmentation showed limited cluster separation, and machine learning models achieved limited predictive performance, indicating that the dataset contains relatively weak predictive signals for purchase prediction.
